# FortiGate-NGFW-Lab

## Overview
Hands-on network security lab implementing a FortiGate NGFW in an enterprise-style environment.
The project focuses on secure access, traffic inspection, and threat prevention.

## Implemented Features
- SSL-VPN (Tunnel & Web Mode) with LDAP integration
- IPsec Site-to-Site VPN
- Firewall Policies and Access Control
- Virtual IPs (VIP) for service publishing
- Web Filtering (Proxy-based)
- DNS Filtering (C&C / Botnet blocking)
- Antivirus and IPS profiles
- Application Control
- Full SSL Inspection

## Validation & Testing
- Verified VPN connectivity and access rules
- Tested policy enforcement and UTM profiles
- Confirmed secure publishing via VIPs
